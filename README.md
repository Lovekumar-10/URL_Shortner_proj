# URL_Shortner_proj  

A simple **URL Shortener** built with **Node.js, Express, MongoDB, and EJS** that allows users to shorten long URLs and access them using a short link.


## 🚀 Features  
- Shorten long URLs into short, easy-to-share links  
- Redirect users to the original URL when accessing the short link   
- Clean and responsive UI with EJS templating  


## 🛠️ Technologies Used  
- **Node.js** – Backend server  
- **Express.js** – Web framework for Node.js  
- **MongoDB** – Database for storing URLs  
- **EJS** – Template engine for dynamic frontend  
- **Mongoose** – ODM for MongoDB
